Recommendations for Improvement

Based on the heuristic evaluation and usability test findings, we propose the following design improvements for the next iteration of ChooZ.

✅ Add a Success Toast Notification

After saving a decision, display a temporary message: “Decision saved!” and automatically return to the Home screen. This provides immediate feedback and reassures the user.

✅ Implement a Visible Close Button on Overlays

Add a clearly labeled “X” or “Close” button to the Random Pick overlay. Also allow tapping outside the overlay to close it, but ensure the primary close control is obvious.

✅ Include a Category Color Legend

Add a small legend on the Home screen (e.g., below the Today’s Overview card) showing each category name and its associated color chip. This eliminates confusion and improves learnability.

✅ Reduce Number of Required Fields

Make the “Notes” field optional and move it to the bottom. Consider combining “Options Considered” and “Chosen Option” into a single selection interface to speed up logging.

✅ Onboarding Tip for Random Pick

During first-time use, show a one‑time tooltip explaining that the Random Pick button can help when indecision strikes.
